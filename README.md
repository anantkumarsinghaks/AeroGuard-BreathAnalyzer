# AeroGuard-BreathAnalyzer
Team Name: AeroGuard
The AeroGuard-BreathAnalyzer is a predictive intelligence engine designed to eliminate the "Data Blindspot" in urban air quality monitoring by providing continuous, hyper-local insights across the entire Delhi-NCR region. To run this project locally, users should first clone the repository and install the necessary dependencies using the command pip install -r requirements.txt, which includes essential libraries like Scikit-Learn, Folium, and SciPy. Once the environment is set up, the user can execute the aeroguard.ipynb Jupyter Notebook to trigger the full data pipeline, which processes historical environmental data and generates the final interactive visualization. The primary output is an index.html file that can be opened in any standard web browser to view the real-time AI-generated heatmap and professional AQI risk legend.
The core logic of the system is built on a dual-engine architecture comprising predictive forecasting and spatial interpolation. We utilize a Random Forest Regressor to analyze historical PM2.5 levels alongside weather variables to forecast air quality 12 hours into the future, ensuring citizens can plan their commutes safely. To address areas without physical sensors, the system employs Inverse Distance Weighting (IDW) logic, which mathematically estimates AQI for any given coordinate by calculating a weighted average from the nearest known monitoring stations. Finally, these discrete predictions are transformed into a smooth, continuous surface using a high-radius heatmap visualization that clearly distinguishes between "Safe Green Corridors" and "High-Risk Red Zones," providing a comprehensive tool for urban environmental awareness.

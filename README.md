# AeroGuard-BreathAnalyzer
Team Name: AeroGuard
Technical Summary: AeroGuard is a predictive intelligence engine designed to eliminate the "Data Blindspot" in Delhi-NCR by providing hyper-local air quality insights where sensors are absent. Leveraging a Random Forest Regressor for 12-hour future forecasting and Inverse Distance Weighting (IDW) for spatial interpolation, the system estimates AQI for every coordinate across the region. The solution culminates in an interactive Continuous Coverage Map that replaces discrete data points with a smooth, color-coded heatmap and a professional AQI Risk Legend, allowing users to visualize safe travel corridors and high-risk "Red Zones" in real-time.
